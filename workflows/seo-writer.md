# AI SEOライター for note

// turbo-all

## 手順

1. ユーザーにメインキーワードを質問してください。
   「記事のメインキーワードを入力してください（例: 仮想通貨 初心者）」

2. `search_web` ツールで以下の2つの検索を実行し、上位5記事のURLを収集してください。
   - 検索クエリ1: `{keyword} site:note.com`
   - 検索クエリ2: `{keyword} SEO 書き方`

3. 収集した上位5記事について `read_url_content` ツールで各記事の内容を取得してください。
   - 取得できない記事はスキップして構いません
   - 最低3記事は取得してください

4. 取得した記事を分析し、以下を抽出してください:
   - 共通する見出し構成パターン
   - 頻出トピック・キーワード
   - 結論の傾向
   - 読者の検索意図（何を知りたいのか）

5. 分析結果を基に **構成案** を作成し、ユーザーに確認してください:
   - タイトル案（SEOキーワードを含む）
   - 見出し（h2）: 3〜5個
   - 各見出しの概要（1〜2行）
   ユーザーが修正を求めた場合は修正してください。

6. 承認された構成案に基づき、note記事を生成してください:
   - 文体: ですます調
   - 文字数: 2000〜3000文字
   - 独自性: 参考記事のコピーやリライト禁止。完全に新しい文章を生成
   - SEO: キーワードを自然に配置（タイトル、最初の段落、見出しに含める）
   - 構成: 導入文 → 各見出しの本文 → まとめ
   - 記事末尾に以下を追記:
     `⚠️ この記事はAIによって生成されました。内容のファクトチェック（事実確認）を必ず行ってください。`

7. `generate_image` ツールでアイキャッチ画像を生成してください。
   - 記事のテーマ・雰囲気に合った画像
   - ImageName は `{keyword}_eyecatch` 形式（スペースはアンダースコアに変換）
   - .agent/rules/nanobanana-image-generation.md のルールに従いNanobanana2形式でプロンプトを作成

8. 以下のコマンドで記事をファイル出力してください:
   ```powershell
   python D:\AG1\ai-seo-writer\scripts\export_article.py --keyword "{keyword}" --article "{記事のMarkdownテキスト}" --image "{生成された画像のパス}"
   ```
   ※ 記事テキストが長い場合は、一時ファイルに書き込んでからパスを渡してください:
   ```powershell
   python D:\AG1\ai-seo-writer\scripts\export_article.py --keyword "{keyword}" --article-file "{一時ファイルパス}" --image "{画像パス}"
   ```

9. 生成結果をユーザーに報告してください:
   - 出力先ファイルパス
   - 記事の文字数
   - 構成の概要
   「記事の内容を確認し、必要に応じて編集してください。」と伝えてください。
